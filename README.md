## Next.js BBS Practice
掲示板にコンテンツを投稿するアプリです。

## Tech Stack
- TypeScript
- Next.js 14(App Router)
- Supabase
- Prisma
- React Hook Form
- Zod
- Tailwind CSS
- shadcn/ui

## Directory Structure
Featuresディレクトリを採用してます。

## Requirements
.env.exampleを参考に.envファイルを作成してください。

## Installation
```console
$ npm -i
```

## Getting Started
```console

```
